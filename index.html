1. Project Folder Structure
Project folder ka naam yolov8-object-detection rakhein aur is structure ko follow karein:
yolov8-object-detection/ |- main.py |- train.py |- evaluate.py |- app.py |- requirements.txt |- README.md |- .gitignore |- index.html |- data/ | |- images/.gitkeep | `- labels/.gitkeep |- models/.gitkeep `- results/.gitkeep
2. Requirements
requirements.txt install karne ke liye:
pip install -r requirements.txt
Pehli run par YOLO model download ho sakta hai, is liye internet connection helpful hoga.
3. Run Commands
Virtual environment
python -m venv venv venv\Scripts\activate
Webcam detection
python main.py --mode webcam --model yolov8n.pt --conf 0.5
Image detection
python main.py --mode image --source data/images/test.jpg --conf 0.5
Video detection
python main.py --mode video --source sample.mp4 --conf 0.5
Training
python train.py --data coco128.yaml --epochs 50
Custom dataset training ke liye apna YAML path dena hoga. GPU recommended hai.
Evaluation
python evaluate.py --model yolov8n.pt --data coco128.yaml
Streamlit app
streamlit run app.py
